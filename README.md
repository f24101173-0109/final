都市量體生成系統
Design Computation & Computational Geometry｜期末專題

本專題以 Python 與 Computational Geometry 建立一套由 GeoJSON 基地資料出發，結合都市計畫規則，自動批次生成 2.5D 建築量體
的運算與視覺化流程。系統可快速比較不同規則假設下的都市量體結果，並輸出CSV（數據）＋PNG（視覺化）＋OBJ（3D 模型）。

●系統功能
1.批次處理、多基地自動生成
2.建蔽率＋容積率雙重控制
3.能處理未預先處理乾淨的實務資料格式
4.清楚區分「規則假設（A）」與「實際法規（B）」
5.幾何計算 × 都市計畫邏輯整合

●都市計畫應用意義
本系統示範了一種以基地幾何與都市計畫管制條件（建蔽率、容積率）為核心的批次量體生成方法。
透過將抽象法規條文轉譯為可運算的幾何流程，系統能快速模擬不同規則假設下的都市空間結果。
此方法可作為都市計畫初期的量體推估、方案比較與法規影響評估工具，協助規劃者在設計前即掌握整體發展強度與空間分布。
相較於逐棟手動建模，本系統提供一種可重複、可調整、可擴充的計算式分析流程。
